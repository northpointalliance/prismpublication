Drop your demo ad images in this folder.

Default filenames used by the demo page:
- `nikeairforce1.jpg`
- `sushinewyork.avif`
- `florista.webp`

You can replace these files with your own images (PNG/JPG/SVG) and keep the same names.
If you use different filenames, update the ad constants in `src/pages/Demo.tsx`:
- `nikeDemoAd.imageUrl`
- `sushiDemoAd.imageUrl`
- `flowerDemoAd.imageUrl`
